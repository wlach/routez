This page is meant to help users encountering some common problems.

Having trouble entering origin / destination

In order to plan a route for you, hbus needs to be able to map the location you specify into a latitude/longitude pair (this process is called geocoding).

At this time, the functionality is very much in development. We don't understand any location other than a specific address (e.g. 5000 North Street, North & Agricola). In particular, we do not understand landmarks (e.g. Scotia Square).

The good news? This will soon be fixed. We plan to add a comprehensive set of landmarks from the OpenStreetMap dataset, as well as dynamic keyboard autocompletion of location. It's going to be awesome.

If you are entering an address that you think should work and doesn't, please let us know at our feedback page.

My route has too much walking!

It used to be that this was due to the data being incomplete. Now that hbus.ca uses the official Metro Transit GTFS, we no longer have that excuse. Please report any issues with this on our feedback page.
