Motivation

The aim of hbus.ca is to make bus travel in Halifax and the surrounding regions less painful. Metro Transit provides reasonably reliable service, but it can be difficult to figure out which bus to take to get from place to place, especially for those who are new to the city or who do not ordinarily take public transit. It should be noted that hbus.ca is NOT affiliated with or endorsed by the city of Halifax or Metro Transit. It is an independantly run site, very much in development at the moment. As such, the directions given by hbus.ca come with zero guarantees.

Technology

hbus.ca was built on top of many technologies, some proprietary, some open source.

The main software that drives this site is called "Routez". It is built on top of the Django framework, with lots of JavaScript code in the front end. The giant map you see is courtesy of the CloudMade Maps API as well as the tireless work of amazing OpenStreetmap volunteers. The actual pathfinding is done via the opensource libroutez library.

Routez was built by the Navarra Group, a team of highly skilled software developers. Want to build an awesome dynamic website or mobile application? Maybe we can help.

Data

To generate a set of data that can be used by this site's pathfinding algorithm, we have prepared a Google Transit Feed of Metro Transit's transit network. The schedule data was converted from the main Metro Transit web site. Currently only the 1, 3, 4, 5, 6, 7, 9, 10, 14, 17, 18, 2, 20, 21, 23, 52, 58, 80, 81 are covered (service off the main Halifax peninsula may be limited to time point stops). Stop location information was gathered using a handheld GPS and/or derived from the GeoBase dataset. The feed and the tools used to build it are opensource and in active development: for more information, see the Halifax Transit Feed project.

It should be noted that Metro Transit had no involvement in the preparation of this data and should not be blamed for any errors or omissions.
